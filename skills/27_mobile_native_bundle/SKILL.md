# 27 Mobile & Cross-Platform

This bundle provides workflows for building high-performance native and hybrid mobile applications.

## Sub-Skills Included
These skills are located in `./sub-skills/`:
- **Native**: `swift-ios-app-architecture`, `kotlin-android-development`, `jetpack-compose-ui`
- **Cross-Platform**: `flutter-state-management`, `react-native-performance`, `capacitor-bridge-plugins`
- **Release**: `play-store-deployment`, `app-store-connect-automation`, `fastlane-deployment-pipeline`

## Usage
Reference any sub-skill by navigating to `./sub-skills/[skill-name]/SKILL.md`.
