# 9.5.1 Automation Lowcode Bundle

This bundle provides workflows for connecting systems and automating repetitive tasks without complex code.

## Sub-Skills Included
These skills are located in `./sub-skills/`:
- **Platforms**: `zapier-make-patterns`, `airtable-automation`, `n8n-workflows`
- **Browser**: `puppeteer-scraping`, `selenium-testing`, `browser-automation-patterns`
- **Productivity**: `slack-app-development`, `discord-bot-patterns`, `notion-api-integration`

## Usage
Reference any sub-skill by navigating to `./sub-skills/[skill-name]/SKILL.md`.
