# 9.4.1 Devops Cloud Bundle

This bundle provides workflows for infrastructure, deployment, and cloud scale.

## Sub-Skills Included
These skills are located in `./sub-skills/`:
- **Containers**: `docker-multi-stage-build`, `kubernetes-deployment-patterns`, `helm-charts`
- **IaC**: `terraform-modules`, `ansible-automation`, `pulumi-infrastructure`
- **CI/CD**: `github-actions-workflow`, `jenkins-pipeline`, `gitlab-ci-cd`

## Usage
Reference any sub-skill by navigating to `./sub-skills/[skill-name]/SKILL.md`.
